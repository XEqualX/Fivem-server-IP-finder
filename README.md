# Fivem-Ip-grabber
A simple IP grabber for fivem. all it needs is the connect code. It uses a custom API to fetch server info like the resources, IP, player count and a bit more. it uses libcpr/cpr.h for networking and nlohmann/json.hpp for json handling. 

This is my first project on github, may be some bugs but error handling is for larps.
